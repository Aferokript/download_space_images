# Космический Телеграмм
Программа для скачиваня фото с spaceX и NASA, а так же для опубликования в телеграмме канале с помощью бота 

## Как установить

Python3 уже должен быть установлен. Затем используйте pip (или pip3, если есть конфликт с Python2) для установки зависимостей:

```pip install -r requirements.txt``` Это установит все зависимости. 

1. Вы должны перейти в главного бота телеграмма: 'Botfather'. Нажимаем на /start и создаем нового бота. Мы получаем от бот
фазера токен от нашего бота. Токен выглядит примерно так: 5665864966:AAGGddDJDFJDUFyry9-cy5UJRLWHjVxTriO8ZVd1Ws
2. Создаем файл .env рядом с space_telegram и в ней переменную окружения с нашим токеном.
3. Для работы скрипты нужны переменные окружения. Без них скрипт не запустится. Нам нужны две: одна для токена НАСА,
вторая для телеграмм-токена. Создаются они так: ПЕРЕМЕННАЯ=ЗНАЧЕНИЕ. 1. TELEGRAM_TOKEN=ВАШ_ТОКЕН, 2. NASA_API_KEY='ВАШ_
КЛЮЧ. (если не хотите регистрироваться, можете подставить в значении DEMO_KEY)

### Примеры запуска

## APOD (Astronomy Picture of the Day):

```python space_telegram.py --source apod```

## EPIC (Earth Polychromatic Imaging Camera):

```python space_telegram.py --source epic```

## SpaceX (фото запусков):

```python space_telegram.py --source spacex```

## Конкретный запуск по ID
```python space_telegram.py --source spacex --id 5eb87d47ffd86e000604b38a```

## Отправить сообщение 
```send_photo(Токен от бота, чат-айди с @, путь к файлу)```
